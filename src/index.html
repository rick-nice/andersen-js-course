To start game press MINE to find some ores. Found ores will appear in your inventory.

Next you need to create a recipe. You can do this by pressing '+' button in your RECIPE inventory. Your RECIPE inventory will be replaced with NEW RECIPE window.

To create new recipe, you need to drop some items to NEW RECIPE window slots and write name of recipe. After creating, new recipe will appear in your RECIPE inventory. Note: new item, created with this recipe will has same name.

Only recipes can be placed in RECIPE inventory.

To FORGE item you need to drop recipe to a recipe big slot at CRAFT space, and drop requred items to slots below. New item will appear in your INVENTORY Only items and ores can be placed in INVENTORY.

If you want to delete any ore, item or recipe you can drop it to TRASH ZONE. You can open info window with button.

To create new recipe, you need to drop some items to NEW RECIPE window slots and write name.
